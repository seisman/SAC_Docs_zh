khronhite
=========

概要
----

对数据应用 Khronhite 滤波器

语法
----

.. code-block:: console

    KHRONHITE [v]

输入
----

v
    截断频率，单位 Hz

缺省值
------

.. code-block:: console

    khronhite 2.0

说明
----

此低通滤波器是两个级联的四阶 Butterworth 低通模拟滤波器的数字近似。
其拐角频率为 0.1 Hz 以增强区域地震的基模瑞利波（Rg）的振幅信号。

头段变量
--------

depmin、depmax、depmen
