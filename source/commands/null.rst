null
====

概要
----

控制空值的绘制

语法
----

.. code-block:: bash

    NULL [ON|OFF|value]

输入
----

value
    将数据中的空值替换为 ``value``

ON
    打开绘图时的空值选项

OFF
    关闭绘图时的空值选项

缺省值
------

.. code-block:: bash

    null off

说明
----

有些情况下，数据中存在间断，此时没有有用的值，称为空值。在多数情况下将
空值设置为一个预定的值，一般来说是0.0、-1.0、-99。通常用户不希望这些值
在绘图时显示。\ ``null`` 命令允许用户定义空值并在绘图时不连接这些点。

示例
----

设置空值为-1.0并打开空值选项：

.. code-block:: bash

    SAC> null on -1.0
