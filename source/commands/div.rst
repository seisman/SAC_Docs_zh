div
===

概要
----

数据文件的每个数据点除以同一个常数

语法
----

.. code-block:: console

    DIV [v1 [v2 ... vn] ]

输入
----

v1
    第一个文件要除以的常数

v2
    第二个文件要除以的常数

vn
    第n个文件要除以的常数

缺省值
------

.. code-block:: console

    div 1.0

说明
----

参见 :doc:`/commands/add`  的相关说明。

头段变量改变
------------

depmin、depmax、depmen
