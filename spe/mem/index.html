mem¶

概要¶

利用最大熵方法计算谱估计

语法¶

MEM [Order n] [Number n]

输入¶

ORDER n

设置预测误差滤波器的时滞阶数为n

NUMBER n

设置用于谱估计的点数

缺省值¶

mem order 25

说明¶

该命令实现了最大熵谱估计法，该方法使用一个预测误差滤波器对数据进行白化处理，得到的谱估计正比于滤波器的能量频率响应的倒数。用户可以自由选择预测误差滤波器的阶数，详情参考 plotpe 命令。

该方法的主要优点是用相对少量的数据即可获得相当高的分辨率，它的缺点是跟传统方法相比没什么理论好说。
