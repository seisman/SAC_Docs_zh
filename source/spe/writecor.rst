writecor
========

概要
----

写一个包含相关函数的 SAC 文件

语法
----

.. code-block:: console

    WriteCOR [file]

输入
----

file
    要写入的 SAC 文件名

缺省值
------

.. code-block:: console

    writecor cor

说明
----

由此命令写出的相关函数的结构取决于用来计算它的算法。由于数据被分配到几个
窗口中，并且样本相关函数是从每一个窗口算出的然后再去平均，则相关函数的
长度由数据窗口的尺寸决定，它实际上容纳的样本数则为数据窗内数据的二倍减
一。然而，由于计算样本相关函数时使用了 FFT 算法，文件中的样本数是2的乘方。
也就是说实际上是数据窗口尺寸的2倍。附加的样本都是零。相关函数也是在文件内
循环回转的。这是由于使用 FFT 算法计算相关函数的特殊性。这意味着零滞后样本是
文件在中的第一个样本，负滞后的样本紧随着正滞后的样本。
