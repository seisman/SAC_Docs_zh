sub
===

概要
----

数据文件的每个数据点减去同一个常数

语法
----

.. code-block:: console

    SUB  [v1 [v2 ... vn]]

输入
----

v1
    从第一个文件中减去的常数

v2
    从第二个文件中减去的常数

vn
    从第n个文件中减去的常数

缺省值
------

.. code-block:: console

    sub 0.0

说明
----

参见 :doc:`/commands/add` 的相关说明。

头段变量改变
------------

depmin、depmax、depmen
