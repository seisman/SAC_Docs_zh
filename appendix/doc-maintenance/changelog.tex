\section{文档更新历史}
\subsection*{2012-01-08 1.0版}
\begin{itemize}
\item 第一版发布，由DOC转换为PDF
\item 参考《数字地震波形分析》一书，翻译了大部分官方文档中的内容
\item 结合SAC 101.4版本，增加、删除和修改了一些命令
\item 增加了书签，方便定位，支持全文搜索
\end{itemize}
